# Dev Log

## What It Is
A web app to log bugs you hit and things you learn while coding — so nothing gets forgotten.

## The Problem It Solves
I kept losing track of bugs I'd already solved and concepts I'd just learned, forcing me to re-google the same things repeatedly. This app gives me one place to log everything with context, so I can actually reference it later.

## What I Intentionally Excluded
- **User authentication** — adding login would require sessions or JWT, significantly increasing complexity for an MVP I only need one person to use.
- **Search/filter** — useful eventually, but out of scope for the MVP; all four CRUD operations work without it.

## Tech Stack
- Backend: Node.js + Express + MongoDB Atlas (Mongoose)
- Frontend: Vanilla HTML + JS
- Deployed: Render (backend) + Netlify (frontend)
